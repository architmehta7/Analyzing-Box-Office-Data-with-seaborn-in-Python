# Analyzing-Box-Office-Data-with-seaborn-in-Python
Hey moviebuffs,want to know which movie was a hit and which one a flop.The data-set used was extracted from TMDB(The Movie DataBase)
Libraries used:pandas,numpy,matplotlib,seaborn,scikitlearn
A relation was drawn b/w movie revenue and budget to see if a movie was a BO hit or flop.Does having an official homepage affect revenue?Distribution of languages in a film.The most frequent words used in movie titles and description were plotted using wordcloud,which words had a negative viewership impact were also analysed using sci-kit learn.
